Responsively-sized <iframe>

CSS の frame-sizing: content プロパティを使うと、 <iframe> 要素のサイズを埋め込み先ドキュメントの レイアウト内在サイズ（layout intrinsic size）に合わせられます。 これによって iframe 内のスクロールバーを出さずに、親ページと シームレスに見えるように埋め込めます。

仕様: CSS Box Sizing Module Level 4 §5.3 / Chrome Platform Status: Responsively-sized <iframe>

→ この機能がなかった時代の従来ハック（postMessage など）はこちら

1. frame-sizing: content（レスポンシブ）

高さは auto のまま、埋め込みドキュメントの内容に応じて iframe の高さが自動で決まります。内側でボタンを押して内容を増減させると、 子ドキュメントが window.requestResize() を呼び出して 高さが再計算されます。

2. 従来の <iframe>（比較用）

同じ内容でも frame-sizing を指定しない場合は固定高さのままで、 内容がはみ出すとスクロールバーが表示されます。
